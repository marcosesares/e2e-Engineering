### Skill Content (SKILL.md) changes → impacts:

| Impacted Component | Reason |
|-------------------|--------|
| `CLAUDE.md` | 🔴 DIRECT — CLAUDE.md mirrors SKILL.md content. Any guideline change must be manually propagated. |

## High-Risk Change Scenarios

| Scenario | Components requiring update | Risk without update |
|----------|-----------------------------|---------------------|
| Modify a behavioral rule in SKILL.md | CLAUDE.md, SKILL.zh.md | Silent divergence — some users follow outdated guidelines |
| Bump version number | plugin.json AND marketplace.json | Marketplace shows stale version |
| Rename plugin or marketplace ID | plugin.json, marketplace.json, README.md | Install commands documented in README break |
| Add a 5th principle | SKILL.md, CLAUDE.md, SKILL.zh.md, README.md, EXAMPLES.md | Partial documentation — new principle undocumented |
| Change skills[] path in plugin.json | Skill loader breaks for all plugin installs | Complete install failure |

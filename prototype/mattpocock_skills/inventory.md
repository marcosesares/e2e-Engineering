## Project Overview

🟢 **skills** is a content/tooling repository — a curated collection of Claude Code agent skills (slash commands and behaviors) authored by Matt Pocock. There is no traditional application runtime; the "product" is the set of Markdown instruction files consumed by AI agents.

Distributed via [skills.sh](https://skills.sh/mattpocock/skills) and installable with:
```bash
npx skills@latest add mattpocock/skills
```

## Skill Anatomy

Every skill is a folder with a required `SKILL.md`. Larger skills may include:

```
skill-name/
├── SKILL.md           # YAML frontmatter (name, description) + Markdown instructions
├── *.md               # Supporting reference files (DEEPENING.md, LOGIC.md, etc.)
└── scripts/           # Utility scripts (bash, node) for deterministic operations
```

`SKILL.md` frontmatter:
```yaml
---
name: skill-name
description: <≤1024 chars — what it does + "Use when [triggers]">
---
```
